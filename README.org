* Tooling

This project uses:

- *GitHub*: gh CLI
- *Editor*: Emacs with org-mode, org-babel
- *Languages*: Clojure (via babashka), Python

* Development Workflow

** Proposal Development

#+begin_src bash
# Start new proposal
jj new -m "WIP: New CFP proposal"
emacs cfp/proposals/01-literate-programming.org

# Tangle code examples
C-c C-v t   # In Emacs: org-babel-tangle

# Generate diagrams  
C-c C-c     # In Emacs: on mermaid src block

# Commit when ready
jj describe -m "Complete proposal draft"
#+end_src

** Review Process

#+begin_src bash
# Create review branch
jj branch create cfp/review-proposal-1

# Push for feedback
jj git push --branch cfp/review-proposal-1
gh pr create --title "Review: Proposal 1"

# Iterate based on feedback
jj new -m "Address review comments"
#+end_src
